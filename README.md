# ndn-mobile

Mobile NDN node for the [ndn-rs](https://github.com/Quarmire/ndn-rs) ecosystem,
plus `ndn-boltffi` — the FFI layer behind the Android apps (Anchor / Ripple).
Builds on [ndn-ext](https://github.com/Quarmire/ndn-ext) for radio/BLE faces.

Part of the [ndn-rs](https://github.com/Quarmire/ndn-rs) ecosystem.
